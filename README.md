# AI-модель для бесед вида One-to-One (1:1)

Этот проект представляет собой персонального голосового ассистента с работоспособной и распространенной архитектурой (Архитектура с центральным агентом: Single-Agent Architecture with Tools). В него входит: 
- **orchestrator**: маршрутизатор или классификатор.
- **user_input_text** и **user_input_stt**: получение текстовых входных данных пользователя напрямую и с помощью распознания речи.
- **general**: основой агент для генерации текста с помощью LLM.
- **tts**: синтез речи по тексту, который сгенерировала LLM.
 
## Установка

> [!IMPORTANT]
> **Обязательные требования:**
> **OS** = **Windows;**
> **RAM** = **16+ GB.**

1. Клонируйте реопзиторий
```bash
git clone https://github.com/KotingGG/Lerche.git
cd Lerche
```

2. Зависимости:
- [Python](https://www.python.org/downloads/)
- Для запуска локальной LLM [Ollama](https://ollama.com/download)
- для удобства и быстроты устанавливки зависимостей `uv`:
```bash
pip install uv
```
- Для установки всех зависимостей из lock-файла:
```bash
uv sync
```

3. Установка локальной модели. [Почему модель `qwen2.5:7b-instruct-q4_0?`](#почему-модель-qwen257b-instruct-q4_0):
```bash
ollama pull qwen2.5:7b-instruct-q4_0
```


## Использование

```bash
uv run main.py
```

## Структура проекта

- `main.py`: Скрипт для запуска модели.
- `graph/`: Логика langgraph агентов. Распознание речи, синтез речи и т.д.
- `confing/`: Конфигурация моделей. Параметры (температура и т.д.), название модели, системный промт.
- `vosk_models/`: Локальная модель для распознания речи. Установлена с [официального сайта vosk](https://alphacephei.com/vosk/models).

## Почему модель `qwen2.5:7b-instruct-q4_0`?
Более подробно про модель: https://ollama.com/library/qwen2.5:7b-instruct
Используется лёгкое квантование для оптимизации. Плюсы:
- Соответствие 16 GB RAM
- Многоязычную поддержку для более чем 29 языков
- Долгие беседы (до 128 тысяч токенов)
- Значительно большим количеством знаний.
- Более устойчив к разнообразным системным подсказкам, улучшая ролевые игры и настройки условий для чат-ботов.
